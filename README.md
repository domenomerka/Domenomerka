Book Tracker — это графическое приложение для управления списком прочитанных книг. Программа позволяет:

Добавлять книги с указанием названия, автора, жанра и количества страниц, просматривать список всех добавленных книг в удобной таблице, фильтровать книги по жанру и по количеству страниц (больше заданного числа), удалять книги из списка, автоматически сохранять данные в файл books.json (данные сохраняются между запусками)

Требования:
- Python 3.6 или выше
- Модуль `tkinter` (входит в стандартную библиотеку Python)

Запуск:
1. Скачайте файл `book_tracker.py`
2. Откройте терминал/командную строку в папке с файлом
3. Выполните команду:
   ```bash
   python book_tracker.py

- Или просто дважды кликните по файлу book_tracker.py, если Python настроен корректно.
